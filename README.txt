Picture Viewer
==============

This is a Common Lisp port of the Microsoft .Net Picture Viewer
Tutorial [1] implemented using Edi Weitz's RDNZL .NET layer for Common
Lisp [2].  It has only tested with International Allegro CL Free
Express Edition 8.2 [3] on 64-bit Windows 7.

Picture-Viewer comes with no warranty of any kind.
